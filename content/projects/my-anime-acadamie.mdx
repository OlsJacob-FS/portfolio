My Anime Academia is a full-stack application created using React.js, powered by
Node.js with Express. The application is a great way for users to discover new
anime they would enjoy. Using the Jikan API to fetch anime data and display it to
users. Users can also use a built-in search feature to search for anime
titles.

## My Github Repository!

[Github Repository](https://github.com/OlsJacob-FS/my-anime-academia)

## Features

- **Discovery**: Discover new anime titles to watch.
- **Search**: The search feature allows users to search anime by title. While
  searching for a title, the search feature will return a list of every anime
  title with a similar name.

- **Random anime generator**: If users are unsure of what to watch or search
  for, they can generate 25 random anime titles by clicking the button at
  the bottom of the home page.

- **Stash**: A built-in feature allowing users to store anime titles they're
  interested in. After users watch the anime they add to their stash, they
  can remove it by clicking the remove button.

- **Forms**: A feature that allows users to post topics related to current
  anime they're watching, interested in, or dislike.

## Technologies

- **Vite/React.js**: A React framework for building static and dynamic websites.
- **Node.js/Express**: A JavaScript runtime environment and web application framework.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Nodemon**: A command-line tool that automatically restarts a Node.js
  application when it detects changes to files in the project directory.
- **MongoDB**: A NoSQL database used to store data from the application.

## Conclusion

My Anime Academia is a simple but effective application to discover new anime
titles while also providing a method to save titles that users are interested
in.
