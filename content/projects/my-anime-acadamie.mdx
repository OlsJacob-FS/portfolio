My Anime Academia is a full-stack application created using ReactJs, powered by
NodeJs with Express. The application is a great way for the user to discover new
anime's they would enjoy. Using the Jikan API to fetch anime data to display to
the user. The user may also use a built in search feature to search for anime
titles.

## Features

- **Discovery**: Discover new anime titles to watch.
- **Search**: A built in search feature allows the user to search anime by
  title.
- **Random anime generator**: The user can generator 25 random anime titles by
  clicking the button at the bottom of the home page.
- **Stash**: A built in feature allowing users to store anime titles they're
  interested in.

## Technologies

- **Vite/ReactJs**: A React framework for building static and dynamic websites.
- **NodeJs/Express**: A file format that allows you to write JSX in your
  Markdown files.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Nodemon**: A command-line tool that automatically restarts a Node.js
  application when it detects changes to files in the project directory.
- **MongoDB**: A noSQL database used to store data from the application.

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone
cd server
npm install
cd my-anime-Academianpm install
```

Once the dependencies are installed, you can run the development server:

```bash
cd server
npm run dev
cd my-anime-Academia
npm start
```

Set up MongoDB using a .env file with the following configuration:

```bash
MONGODB_URI=mongodb://127.0.0.1/27017:myanimeacademia
PORT = 3000
```

Server: The development server will start at `http://localhost:3000`. Client:
The development server will start at `http://localhost:5173`.

My anime academia is a simple but effect application to discover new anime
