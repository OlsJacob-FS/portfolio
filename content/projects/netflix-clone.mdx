This project is a Netflix clone application built using React.js, Firebase, and
Redux. The app allows users to browse movies and shows after setting up their
login information.

## See it for yourself

[Netflix Clone Application](https://jolsfsdas1.firebaseapp.com/)

[Github repository](https://github.com/OlsJacob-FS/spotifyApi)

## Features

- **User Authentication**: User's will be able to setup up a profile using their
  email and password of choice.

## Technologies

- **Frontend**: React.js, React Router, Redux
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Styling**: CSS, Styled Components
- **Deployment**: Firebase Hosting

## Conclusion

This Netflix clone application is my first web application hosted and managed by
Firebase. Using firebase brought several challenges for me to overcome. Reading
through their documentation I was successfully able to set up authorization,
storing user data, and hosting.
